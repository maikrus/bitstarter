Real FAQ Stats idea is to show whether any question that was asked in the Internet is really frequent and how frequent. It collects questions statistics with source links about any topic that present in the Wikipedia. The content should be filled by the Internet crowd. As soon as the content reaches it's critical mass, it would be interesting to provide analysis which question/answer is the most popular in the Internet. Further interesting analysis could be performed later based on the provided community data. Images on this page provides some visual info how the site is supposed to work. The source data numbers are fake, but the picture shows how they should be organized.

How it works:

  • A user chooses a topic from Wikipedia
  • The Wikipedia link is subject to verification
  • The FAQ title gets from the Wikipedia article title
  • A user posts a question and an answer (both found in the Internet) about the topic
  • A user posts a link to a source of the answer
  • The source could be any Internet resource including video, audio, etc. that contains this answer
  • The question and the answer is posted to Real FAQ Stats
  • The more similar answer sources we have the upper location in the list the question has
  • The highest sources count answer is "the most popular answer"
  • The most popular answer is supposed to be a right answer to the question, but not always.
Future development will include robots to collect the content from the Internet.
